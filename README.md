# Responsive Navigation Bar and Sidebar

## Project Overview
This project demonstrates a responsive navigation bar with a sliding sidebar for both desktop and mobile devices. On smaller screens, the top navbar collapses into a hamburger menu. Clicking the menu opens a slide-in sidebar with navigation links and a semi-transparent overlay for better user experience.

The layout is built using HTML, CSS, and JavaScript, ensuring a smooth, modern, and interactive user interface.

## Features
- Responsive top navigation bar
- Hamburger menu for mobile devices
- Slide-in sidebar
- Overlay effect when sidebar is open
- Clean and modern UI
- Fully customizable links and colors

## Tech Used
- HTML5
- CSS3 (Flexbox and Media Queries)
- JavaScript (toggle sidebar)
